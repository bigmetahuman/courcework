[35] Тема: «Обработка больших массивов данных с использованием
возможностей библиотеки Pandas и SciPy.stats.».

a. Найти в интернете или сгенерировать с помощью искуственного
интеллекта входной файл данных в формате csv, содержащий не менее
800 записей с числовыми и категориальными характерисктиками
объектов, в соответствии с выбранной тематикой (например,
параметры, влияющие на стоимость домов или квартир и фактическая
стоимость). В данных должно быть не менее 1% пропусков или
некорректных значений в 2х или 3х столбцах. Процесс получения
исходных данных подробно описать в отчете.

b. Импортировать данные из csv файла в объект DataFrame библиотеки
Pandas.

c.Выполнить первичный анализ данных: определить количество
объектов, данные о которых представлены в файле, количество
числовых и количество категориальных параметров, описывающих
каждый объект.

d. Изучить входные данные на предмет наличия пропущенных значений
или некорректных данных.

e. Изучить методы устранения (или заполнения) пропусков во входных
данных и применить эти методы для «коррекции» входного массива
данных. На выходе должен получиться объект DataFrame, не
содержащий некорректных или пропущенных значений.

f.Определить для каждого признака (для которого это возможно)
основные описательные статистики (среднее, дисперсию, медиану и
т. п.), используя встроенные возможности библиотеки Pandas. Пояснить
в отчете смысл вычисленных значений.

g. Визуализировать статистические распределения каждого признака,
используя возможности библиотек Matplotlib (и/или Seaborn). Пояснить
смысл полученных графиков распределений.

h. На основании проведенного анализа выделить в исходных данных 3
признака, которые имеют наибольшее влияние на формирование
значения определяющего выходного параметра для каждого из
объектов (например, цены дома). Обосновать выводы в отчете.
